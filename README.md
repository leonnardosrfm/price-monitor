Monitor de preços em Python que acompanha produtos em e-commerces e envia alertas no Discord quando ocorre queda de valor, utilizando seletores CSS configuráveis.

Funcionalidades
  Monitoramento periódico de preços
  Extração de preço via CSS selector
  Histórico de preços em SQLite
  Alertas no Discord via webhook
  Cooldown para evitar notificações repetidas
  Configuração por arquivo YAML

Tecnologias  

  Requests
  BeautifulSoup
  SQLite
  PyYAML
  Discord Webhooks
